# AgentFolio

**Vai trò (Role)**: Khám phá Mạng Lưới Agent (Agent Discovery)

Sử dụng kỹ năng này khi bạn muốn **khám phá, so sánh, và nghiên cứu các AI agent** tự động. AgentFolio (https://agentfolio.io) theo dõi các framework, thư viện và sản phẩm tự trị hiện tại.

Giúp bạn:
- Tìm kiếm các agent có sẵn thay vì xây cái mới.
- Nắm bắt bối cảnh chung.
- Tìm benchmark để đánh giá.

## Năng Lực (Capabilities)
- Tìm Agent, tool và framework theo Use Case.
- So sánh các Agent (UX, API, chức năng).
- Phát hiện các ngách thị trường còn bỏ sót.

## Cách sử dụng (How to Use)
Vào trang `https://agentfolio.io` để lấy danh sách agent. Phân tích chức năng và giá trị gốc (Core Promise) của nó.

## Khi Nào Sử Dụng (When to use)
Khi lập kế hoạch xây Agent/Workflow mới và cần tìm hiểu các thiết kế UX/Pattern đang thịnh hành.
