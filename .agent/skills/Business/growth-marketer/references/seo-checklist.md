# SEO Audit & Checklist

## Technical SEO
- [ ] Robots.txt và Sitemap.xml khai báo chính xác.
- [ ] Xử lý thẻ Canonical tag cho trang trùng lặp nội dung.
- [ ] Mobile-Friendly: UI không bị vỡ trên di động, font chữ dễ đọc.
- [ ] Hreflang Tags cho trang đa ngôn ngữ.

## Content & On-page SEO
- [ ] Keyword Density tự nhiên, tránh nhồi nhét từ khóa.
- [ ] Thẻ Title (50-60 ký tự) và Meta Description (150-160 ký tự) hấp dẫn.
- [ ] Heading Tags (H1, H2, H3) cấu trúc như một mục lục bài học.
- [ ] Alt text cho tất cả hình ảnh.

## Off-page SEO
- [ ] Backlink profile building (Guest post, chia sẻ MXH).
- [ ] Brand mentions trên các trang uy tín.
